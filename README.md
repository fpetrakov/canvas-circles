# canvas-circles

Simple but cool looking animation built with Canvas API

## Stack
- Canvas
- Javascript
- SCSS
- Gulp
- CSS
- HTML

## NPM Commands
| Script  | Description |
| ------------- | ------------- |
| npm start  | Start gulp development server at localhost:3000  |
| npm run build  | Build the application to ./dist directory  |
